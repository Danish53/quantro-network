Placeholder images are SVGs so the About page works before you add final assets.

To replace:
1. Add your files under public/images/about/ (e.g. hero-dashboard.png).
2. Edit app/components/about/aboutImages.js and point each key to your filename.

Suggested filenames:
- hero-dashboard.png / .webp  — hero monitor
- zenith-product.png — Zenith / algorithms visuals
- scalable-thumb.png — small card thumbnail
- mobile-app.png — phone mockup
- atlas-security.png — Security / Atlas visual

You can delete the .svg placeholders after switching paths.
